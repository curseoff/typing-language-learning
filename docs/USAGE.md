# 使い方ガイド

「英文・和文タイピング」の遊び方・操作の詳しい説明です。アプリは次の URL で動きます。

🎮 **デモ**: https://curseoff.github.io/typing-language-learning/

> 概要・収録内容は [README](../README.md)、開発者向けは [docs/DEVELOPMENT.md](DEVELOPMENT.md) を参照してください。

## 基本の流れ

1. ブラウザでデモを開く。
2. スタート画面で **種類 → レベル → モード**（種類により項目は変わります）を選ぶ。
3. **「スタート」**ボタン、または **`Enter`** で開始。**最初の正しい打鍵で計測開始**。
4. 出題に従って入力する。
   - 英文はそのまま入力（大文字・記号も）。
   - 和文は**ローマ字で入力**（画面は漢字のまま、打った分だけ色づく）。例: `彼は公園で走ります。` → `karehakouendehashirimasu.`
   - 4択は、選択肢の語／文を**打つと確定**（クリックでも可）。
5. **打ち間違えると赤くなり、正しく打つまで進めません**（打ち間違いは消さなくてOK）。
6. 終了条件（600文字 or 問題数 or 40打）に達すると**結果画面**へ。記録・ランキングが表示されます。
7. 結果画面の**「もう一度チャレンジ」**で、**まったく同じ問題列**を再挑戦できます（種類により対応）。

## スタート画面の操作

スタート画面は **「行（項目）」×「選択」** の2軸で選びます。

| キー | 動作 |
|---|---|
| `↑` `↓` | **項目（行）を移動**。種類 → 物語/レベル/テーマ/モード → 記録ランキング/収録一覧。**一番下まで行くと先頭へループ** |
| `←` `→` | **その行の中で選択を移動**（端で停止） |
| `Enter` | スタート（結果画面では「もう一度」） |
| `Esc` | プレイ中・結果画面からトップへ戻る |

- **フォーカス中の行**の選択は**青い枠**、それ以外の行の選択は**青い背景**で表示されます。
- マウスでクリックしても選べます（クリックした行にフォーカスが移ります）。
- `←` `→` で「種類」を切り替えると、その下の項目（レベル・モード等）も種類に応じて入れ替わります。

## 種類とモード

### 📝 文章（マラソン）
会話文を打ち、**600文字**で終了。場面と丁寧さでレベルが上がります。
- レベル：日常会話（R1 あいさつ・自己紹介／R2 日常生活／R3 旅行・外出）、ビジネス会話（R4 オフィスの基本／R5 電話・連絡／R6 会議・交渉）
- モード：`英語・日本語`（交互）／`英語`／`日本語`／`英語訳`／`日本語訳`

### 📖 物語
テキストを打って進め、**選択肢を打つと分岐**するサウンドノベル風。**4つのエンド**があり、発見したエンドと記録（速度ランキング）が残ります。選んだモードがそのまま適用されます。
- 物語：海外旅行アドベンチャー／クライミング など（「物語」の行で選択）
- 結果画面では、進んだ本文と**自分が選んだ選択肢**が時系列で表示されます。

### 🔤 単語
語彙学習。難易度4段階（基礎／初級／中級／上級）× テーマ（日常／旅行／ビジネス）。
- 入力モード：`英語・日本語`／`英語`（和訳→英単語）／`日本語`（英単語→和訳）… 600文字で終了
- 4択モード：`英語訳`（和訳→英単語を選ぶ）／`日本語訳`（英単語→和訳を選ぶ）… 30問。**選択肢を打って選ぶ**

### 📚 英英辞典
英語の定義で語彙を学ぶ。
- `単語4択`：英語の定義 → 英単語を選ぶ（回答後に和訳）
- `説明4択`：英単語＋意味 → 合う説明文を打って選ぶ
- `英語入力`：定義文を打つ／`日本語入力`：和訳を打つ

### ⌨️ タッチタイピング
ブラインドタッチ（画面を見ずに打つ）の練習。**40打のドリル**で終了。
- レベル：ホームポジション／上段／下段／数字／すべて
- モード：
  - **やさしい**：打つキーが画面のキーボードで**ハイライト**されます。
  - **むずかしい**：**ハイライトなし**。位置を思い出して打つ本番のブラインドタッチ。
- 画面のキーボードは **指ごとに色分け**（小指/薬指/中指/人差し指）し、F・J にホームポジションの印、上に「使う指」を表示。
- **打つと該当キーが沈み込み**、**正解は緑の枠**、**ミスは押した（誤った）キーが赤い枠**で光ります。
- **記録ランキング**：完走すると速度（打/分）・正確率・時間を**レベル×モード別**に保存し、スタート画面のタッチタイピングに**速い順**で表示します。

## 共通の特徴

- **自由なローマ字入力**：`shi`/`si`、`tsu`/`tu`、`ji`/`zi`、促音 `gakkou`、撥音 `n`/`nn` などどれでもOK。入力した綴りに表示が追従します。
- **日本語は漢字で表示**：ローマ字で打つと、対応する漢字が順に色づきます。
- **4択は打って選ぶ**：選択肢の文字も打った分だけ色づき、進捗がわかります（クリックでも可）。
- **間違えると正しく打つまで進めない**（打ち間違いは消さなくてよい）。
- **文末記号も入力**：英 `.` → 和 `。`／英 `?` → 和 `？`。
- **記録ランキング**：速度（打/分）・時間・ミス・正確率を、**種類・レベル・モードごと**にブラウザ内（localStorage）へ永続化します。
- **収録一覧**：スタート画面で「記録ランキング / 収録一覧」を切り替えると、収録された問題と**問題ごとの記録（練習回数・平均ミス・打/秒）**を確認できます。
- **もう一度チャレンジ**：結果画面から、**同じ問題列**を再現して再挑戦できます。

## ヒント

- まずは **タッチタイピング → ホームポジション → やさしい** で運指に慣れ、慣れたら **むずかしい** や **すべて** に進むのがおすすめです。
- 和文入力で詰まったら、ローマ字の打ち方は1通りではありません（`し`＝`shi`/`si` など）。打ちやすい綴りでOKです。
- 記録は端末（ブラウザ）内に保存されます。別の端末・ブラウザには引き継がれません。
