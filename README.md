# 英文・和文タイピング

日本人のための英語学習タイピングゲームです。**5つの種類**（文章・物語・単語・英英辞典・タッチタイピング）から選び、英語と日本語をタイピングしながら語彙・読解・入力を鍛えられます。和文はローマ字で入力でき、漢字はそのまま表示されます。

🎮 **デモ**: https://curseoff.github.io/typing-language-learning/

React + Vite 製。ブラウザで動きます（Electron で各自ビルドも可）。

## 種類とモード

スタート画面の**タブで種類を選び**、種類ごとのレベル・モードを選んで開始します。

### 📝 文章（マラソン）
会話文を打ち、**600文字**で終了。場面と丁寧さでレベルが上がります。
- レベル：日常会話（R1 あいさつ・自己紹介／R2 日常生活／R3 旅行・外出）、ビジネス会話（R4 オフィスの基本／R5 電話・連絡／R6 会議・交渉）
- モード：`英語・日本語`（交互）／`英語`／`日本語`／`英語訳`／`日本語訳`

### 📖 物語（海外旅行アドベンチャー）
テキストを打って進め、**選択肢を打つと分岐**するサウンドノベル風。**4つのエンド**があり、発見したエンドと記録（速度ランキング）が残ります。選んだモードがそのまま適用されます。

### 🔤 単語
語彙学習。難易度4段階（基礎／初級／中級／上級）× テーマ（日常／旅行／ビジネス）。
- 入力モード：`英語・日本語`／`英語`（和訳→英単語）／`日本語`（英単語→和訳）… 600文字で終了
- 4択モード：`英語訳`（和訳→英単語を選ぶ）／`日本語訳`（英単語→和訳を選ぶ）… 30問。**選択肢を打って選ぶ**

### 📚 英英辞典
英語の定義で語彙を学ぶ。
- `単語4択`：英語の定義 → 英単語を選ぶ（回答後に和訳）
- `説明4択`：英単語＋意味 → 合う説明文を打って選ぶ
- `英語入力`：定義文を打つ／`日本語入力`：和訳を打つ

### ⌨️ タッチタイピング
ブラインドタッチ（画面を見ずに打つ）の練習。
- キーボードを表示し、**打つべきキーをハイライト**＋ホームポジションに印
- **指の色分け**と「使う指」表示で誘導
- レベル：ホームポジション／上段／下段／数字／すべて。40打のドリル

## 共通の特徴

- **自由なローマ字入力**：`shi`/`si`、`tsu`/`tu`、`ji`/`zi`、促音 `gakkou`、撥音 `n`/`nn` などどれでもOK。入力した綴りに表示が追従
- **日本語は漢字で表示**：ローマ字で打つと、対応する漢字が順に色づく
- **4択は打って選ぶ**：選択肢の文字も打った分だけ色づき、進捗がわかる
- **間違えると正しく打つまで進めない**（打ち間違いは消さなくてよい）
- **文末記号も入力**：英 `.` → 和 `。`／英 `?` → 和 `？`
- 速度（打/分）・時間・ミス・正確率を記録。**記録ランキングは種類・レベル・モードごと**にブラウザ内へ永続化
- **収録一覧**：スタート画面で「記録ランキング / 収録一覧」を切り替え、収録された問題と**問題ごとの記録（練習回数・平均ミス・打/秒）**を確認できる

## 使い方

詳しい遊び方・操作・各モードの説明は **[docs/USAGE.md（使い方ガイド）](docs/USAGE.md)** にまとめています。

ざっくりした流れ：

1. ブラウザで[デモ](https://curseoff.github.io/typing-language-learning/)を開く
2. スタート画面で **種類 → レベル → モード** を選ぶ（`↑` `↓`＝項目、`←` `→`＝選択、`Enter`＝スタート、`Esc`＝戻る）
3. 英文はそのまま、和文はローマ字で入力（漢字表示のまま色づく）。4択は選択肢を打って選ぶ
4. 打ち間違えると正しく打つまで進めません。終了すると結果・記録ランキングが表示されます
