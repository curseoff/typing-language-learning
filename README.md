# 英文・和文タイピング

英文と和文（日本語）を交互に打つ、語学学習向けのタイピングゲームです。
英文はそのまま英語スペルで、和文はローマ字で入力します。全文を最初から表示し、600文字を打ち切るとタイムと速度が記録されます。

React + Vite 製。ブラウザでもElectronのデスクトップアプリとしても動きます。

## 特徴

- **英文 → 和文を交互に**連結した文章を、最初から最後まで連続入力
- **英文は英語スペル**、**和文はローマ字**で入力
- **ローマ字は自由**（`shi`/`si`、`tsu`/`tu`、`ji`/`zi`、促音 `gakkou`、撥音 `n`/`nn` などどれでもOK）。入力した綴りに合わせて表示が切り替わる
- **間違えると正しく打つまで進めない**（打ち間違いは消さなくてよい）
- 文末の `。` はピリオド `.` を打つ。文と文の区切りに**空白入力は不要**
- 上部に英語・下部に日本語を常に表示し、**どこまで打ったか**を色で表示
- **600文字で終了**し、速度（打/分）・時間・ミス・正確率を表示
- **問題ごとの記録**（1文ごとの速度とミス回数）を一覧表示
- **記録ランキング**を速い順に最大15件保存（ブラウザ/アプリ内に永続化）

## 動かす

前提: Node.js 18 以上。

```bash
npm install      # 初回のみ
npm run dev      # 開発サーバー起動 → http://localhost:5173
```

ブラウザ（Chrome推奨）で開けば遊べます。

## Electron（デスクトップアプリ）

```bash
npm run electron:dev   # ウィンドウで起動（ホットリロード付き、開発用）
npm run dist:dir       # 動作確認用パッケージを release/ に生成（インストーラ無し）
npm run dist           # 配布用インストーラを release/ に生成
```

- `npm run dist` を実行したOSの形式で出力されます（macOS=`.dmg` / Windows=`.exe` / Linux=`.AppImage`）。
- アプリは**未署名**です。macOSでは初回起動時に警告が出るため、`.app` を右クリック →「開く」で起動してください。
- アイコンは未設定（Electron既定）。`build/icon.icns`(Mac) / `build/icon.ico`(Win) / `build/icon.png`(Linux) を置くと自動採用されます。

## 遊び方

1. 「スタート」を押す
2. 表示された文章を先頭から入力（最初の正しい打鍵で計測開始）
   - 英文の行は英語をそのまま（大文字・記号も入力）
   - 和文の行はローマ字で（例: `彼は公園で走ります。` → `karehakouendehashirimasu.`）
3. 打ち間違えると赤く表示され、正しいキーを打つまで進めません
4. 600文字に到達すると結果画面へ。速度・問題ごとの記録・ランキングが表示されます

## カスタマイズ

出題文は `src/sentences.js` を編集するだけで追加・差し替えできます。

```js
{ en: 'I go to school every day.', ja: '私は毎日学校へ行きます。', kana: 'わたしはまいにちがっこうへいきます。' }
```

- `en`: 英文（そのまま入力対象）
- `ja`: 和文の表示（漢字かな）
- `kana`: 和文の読み（ひらがな）。これを元にローマ字入力を判定する。文末の `。` も含める

その他の調整箇所（いずれも `src/App.jsx` 上部の定数）:

- `TARGET_KEYS` … 終了する文字数（既定 600）
- `MAX_RECORDS` … 保存するランキング件数（既定 15）

## 構成

```
.
├─ index.html            エントリHTML
├─ vite.config.js        Vite設定（base: './' でElectron対応）
├─ electron/
│  └─ main.cjs           Electronメインプロセス（ウィンドウ生成・画面読み込み）
└─ src/
   ├─ main.jsx           Reactエントリ
   ├─ App.jsx            ゲーム本体（出題・入力判定・計測・記録）
   ├─ App.css            スタイル
   ├─ sentences.js       出題文データ
   └─ romaji.js          かな⇄ローマ字エンジン（複数の打ち方を受理／canonical変換）
```

## 技術メモ

- ローマ字判定は `romaji.js` がかな読みから「許容する全ローマ字パターン」を展開して照合（`shi`/`si` などを同時に許容）。表示は標準（ヘボン式）を既定にしつつ、入力に追従して切り替わる。
- 記録は `localStorage`（キー: `typing-records-v1`）に保存。速い順で最大15件。
- 速度 = 文字数 ÷ 経過分（打/分）。問題ごとの速度・ミスは1文単位で計測。
